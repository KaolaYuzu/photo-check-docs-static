HOW IT WORKS · V1.4 PIPELINE

兩個引擎，同時執行

01 上傳圖片
拍照或截圖後上傳，支援 PNG / JPG / WebP，上限 5MB。
02 雙引擎辨識
Google Document AI 辨識文字位置，前端 Canvas 偵測表格線條，結果合併。
03 雙模式輸出
Data Mode 下載 Excel / CSV / JSON，Visual Mode 下載 SVG / PDF，各自獨立邏輯。
