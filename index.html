How It Works

三步驟，文件數位化 Three steps to digitize documents

01
上傳表格照片Upload the photo
把紙本進貨單、報表、菜單拍照或截圖，直接拖入上傳區。 Photograph or screenshot any paper document. Drag it into the upload zone.
02
AI 解析 → SchemaAI parse → Schema
系統呼叫一次 OCR，將辨識結果轉換為本機 Table Schema。辨識完成後不再聯網。 One OCR call converts the result to a local Table Schema. No more network calls.
03
校對並匯出Review & export
在網頁直接修改儲存格，確認後一鍵匯出 XLSX / SVG / PDF / CSV / JSON。 Edit cells in the browser. Export XLSX / SVG / PDF / CSV / JSON in one click.

tableSchema = {
  version: "v2",
  source: {
    imageName, imageWidth, imageHeight,
    ocrEngine: "google-document-ai"
  },
  columns: [ // id, x, width, label, locked
    { id, label, width, locked: false }
  ],
  rows: [ // id, y, height, type
    { id, y, height, type: "header"|"data" }

  cells: [{
    id, rowId, colId,
    text, rowSpan, colSpan,
    confidence, bbox,
    style: { bold, align,
      borderTop|Right|Bottom|Left }
  }],
  meta: {
    createdAt, language,
    reconstructionMode
  }
}
