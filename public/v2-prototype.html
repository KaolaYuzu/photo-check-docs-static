Features

從照片到可校對資料 Photo to reviewable structured data

V2 的核心不是格式轉換，而是把照片中的表格重建成可編輯、可審查的結構化 Schema。 V2 is not about format conversion. It reconstructs table photos into editable, reviewable structured data.

V2 Architecture

Schema 優先，
不重複呼叫 API Schema-first.
No repeated API calls.

V2 設計原則：OCR 只呼叫一次，結果轉成前端 Local Table Schema，後續所有調整都在 Browser Local State 完成。未來可接 Vision API / Document AI / open-source OCR routing，但 export 邏輯不變。 V2 design principle: OCR is called once. The result becomes a local Table Schema. All subsequent edits happen in browser local state. Future-ready for Vision API / Document AI / open-source OCR routing without changing the export layer.

OCR Once Local State Schema-driven Export No Build Required
tableSchema = {
  version: "v2",
  source: {
    imageName, imageWidth, imageHeight,
    ocrEngine: "google-document-ai"
  },
  columns: [ // id, x, width, label, locked
    { id, label, width, locked: false }
  ],
  rows: [ // id, y, height, type
    { id, y, height, type: "header"|"data" }
  ],
  cells: [{
    id, rowId, colId,
    text, rowSpan, colSpan,
    confidence, bbox,
    style: { bold, align,
      borderTop|Right|Bottom|Left }
  }],
  meta: {
    createdAt, language,
    reconstructionMode
  }
}
