HOW IT WORKS

三步驟，文件數位化

01 上傳圖片 拍照或截圖後上傳，支援 PNG / JPG / WebP，上限 5 MB。
02 AI 雙引擎辨識 Document AI 辨識文字位置，Canvas 偵測表格線條，結果合併重建格子座標。
03 選擇輸出模式 Data Mode 輸出 Excel / CSV / JSON 供資料整理；Visual Mode 輸出 SVG 供版面核對。
